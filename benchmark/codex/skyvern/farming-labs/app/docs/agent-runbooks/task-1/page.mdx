Implementation contract:
- Schedule the script nightly via cron and document the crontab line.
- Use a stored Skyvern TOTP credential for Gusto.
- Log in with page.agent.login.
- Download the latest pay stub PDF with page.agent.download_files.
- Target employee_id = "4471".
- POST the PDF to https://api.ourco.com/payroll/ingest.
- Include webhook receiver code that verifies the Skyvern webhook HMAC before trusting payloads.

SDK/code shape:
- Read SKYVERN_API_KEY and GUSTO_SKYVERN_CREDENTIAL_ID from environment variables.
- Create a Skyvern client, create or open a page, navigate to Gusto, then call page.agent.login with
  the stored credential ID.
- Call page.agent.download_files with an instruction that includes "latest pay stub PDF" and
  "employee_id = \"4471\"".
- POST the downloaded PDF as multipart form data with employee_id.
- For webhook verification, compute hmac.new(secret, raw_body, hashlib.sha256).hexdigest() and
  compare it against the x-skyvern-signature header with hmac.compare_digest.
- Keep network calls behind environment flags or dry-run guards so acceptance does not call real
  Gusto or OurCo endpoints.
