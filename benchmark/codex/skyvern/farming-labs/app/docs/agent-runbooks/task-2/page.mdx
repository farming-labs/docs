Implementation contract:
- Read or define a list of 200 Amazon URLs.
- Launch one cloud browser session pinned to Germany.
- Reuse the same page/session while iterating URLs.
- Extract title, price, and stock with page.extract and a JSON schema.
- Cap each URL at max_steps=15.
- First run may train with engine="skyvern-2.0"; subsequent runs should use run_with="code".

SDK/code shape:
- Read SKYVERN_API_KEY and an input file such as AMAZON_URLS_CSV from environment variables.
- Create one Skyvern client and one cloud browser session/page before the loop.
- Use Germany/DE geo targeting on the launched cloud browser session.
- Build a Pydantic or JSON schema with title, price, stock, url, and extracted_at.
- For each URL, navigate the existing page, then call page.extract with the schema and max_steps=15.
- Use engine="skyvern-2.0" and run_with="agent" for the optional warm-up run, then run_with="code"
  for the remaining URLs.
- Write results to JSONL/CSV; keep real browsing behind a RUN_REAL_AUTOMATION guard.
