Implementation contract:
- Automate buy.stripe.com/test_abc123 using only the SDK.
- Fill payment details with a stored credit_card credential.
- Poll or retrieve the 6-digit OTP from no-reply@stripe.com and send it with send_totp_code.
- Pass Cloudflare Turnstile handling through the Skyvern flow.
- Retry the full flow up to 3 times.
- Map failures with error_code_mapping for card_declined and captcha_failed.

SDK/code shape:
- Read SKYVERN_API_KEY and STRIPE_CARD_CREDENTIAL_ID from environment variables.
- Create a Skyvern client and page, then navigate to buy.stripe.com/test_abc123.
- Use a stored credit_card credential for card entry rather than hardcoded card data.
- Poll email for no-reply@stripe.com and extract the 6-digit OTP, then call send_totp_code.
- Let the Skyvern browser flow handle Cloudflare Turnstile; classify Turnstile failures as
  captcha_failed.
- Wrap the full checkout flow in a for attempt in range(1, 4) retry loop.
- Configure error_code_mapping so card_declined and captcha_failed become explicit failure codes.
- Keep external checkout disabled unless RUN_REAL_AUTOMATION=1.

- 3
