# API Reference

The Surge API lets you create deeper, more automated messaging workflows across
accounts, campaigns, contacts, messages, recordings, and embedded experiences.

This Farming Labs version keeps the original Surge structure:

- [Introduction](/api-reference/introduction)
- [Endpoints](/api-reference/endpoint)
- [Webhooks](/api-reference/webhooks)

Use the endpoint pages in this section for request and response details, then
pair them with the webhook pages when you need event-driven updates in your app.
