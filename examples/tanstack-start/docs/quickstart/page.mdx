# Quickstart

Create a new page:

```bash
mkdir -p docs/guides/first-page
```

Then add `docs/guides/first-page/page.mdx`:

```mdx
---
title: "First Page"
description: "A new page in your TanStack Start docs."
---

# First Page

Hello from TanStack Start.
```

The page will be available at `/docs/guides/first-page`.
