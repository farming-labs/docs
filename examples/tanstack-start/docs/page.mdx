# TanStack Start support

This example proves the React docs runtime is no longer tied to the Next.js router.

<MyNote>
  `docs.config.tsx` still controls your theme, navigation, page actions, llms.txt, and registered MDX components.
</MyNote>

## What is working here

- Shared `docs.config.tsx` theme setup
- MDX rendering with registered React components
- Search API at `/api/docs`
- Copy page / open docs actions
- llms.txt generation
