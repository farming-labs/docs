# Installation

Install the packages for the SvelteKit docs example:

```bash
pnpm add @farming-labs/docs @farming-labs/svelte @farming-labs/svelte-theme
```

## Configure Docs

Set `entry`, `contentDir`, `nav`, and `i18n` in `src/lib/docs.config.ts`.
