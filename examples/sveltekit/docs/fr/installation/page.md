# Installation

Installez les packages utilises par l'exemple SvelteKit:

```bash
pnpm add @farming-labs/docs @farming-labs/svelte @farming-labs/svelte-theme
```

## Configuration

Definissez `entry`, `contentDir`, `nav` et `i18n` dans `src/lib/docs.config.ts`.
