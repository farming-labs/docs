# Introduction

Cet exemple SvelteKit utilise l'i18n du framework avec `?lang=`.

- Ouvrez `/docs?lang=en` pour l'anglais.
- Ouvrez `/docs?lang=fr` pour le francais.
- Le selecteur de langue conserve la locale active sur les liens internes.
