# Demarrage

L'exemple Next.js garde l'UI partagee synchronisee avec la locale active, puis
selectionne la bonne page MDX a partir de `searchParams.lang`.

## Verifications

- Ouvrez `/docs/getting-started?lang=en` et `/docs/getting-started?lang=fr`.
- Changez de langue depuis le selecteur dans la sidebar.
- Verifiez que le contenu change avec la locale choisie.
