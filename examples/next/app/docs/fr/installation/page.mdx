# Installation

Installez les packages pour l'exemple de docs Next.js:

```bash
pnpm add @farming-labs/docs @farming-labs/next @farming-labs/theme
```

## Configuration

Activez `i18n` dans `docs.config.tsx`, puis choisissez le bon contenu MDX selon
`searchParams.lang` dans le wrapper de route.
