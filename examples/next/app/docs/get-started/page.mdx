# Getting Started

Get up and running in minutes. This guide will walk you through installation and basic configuration.

<Callout type="info">
  Prerequisites: Node.js 18+ and pnpm (or npm/yarn)
</Callout>

## Installation

Install the docs package and its peer dependencies:

```bash
pnpm add @farming-labs/docs fumadocs-ui fumadocs-core
```

For Next.js projects, you'll also need the MDX loader:

```bash
pnpm add @next/mdx @mdx-js/loader @mdx-js/react
```

## Project Setup

### 1. Create the config file

Create a `docs.config.ts` in your project root:

```ts
import { defineDocs } from "@farming-labs/docs";
import { fumadocs } from "@farming-labs/docs/theme/fumadocs";

export default defineDocs({
  entry: "docs",
  theme: fumadocs({
    ui: { colors: { primary: "#22c55e" } },
  }),
  metadata: {
    titleTemplate: "%s – Docs",
    description: "Your documentation",
  },
});
```

### 2. Add the layout

Create `app/docs/layout.tsx`:

```tsx
import docsConfig from "@/docs.config";
import { createDocsLayout } from "@farming-labs/docs/next";

export default createDocsLayout(docsConfig);
```

### 3. Add your first page

Create `app/docs/page.mdx`:

```mdx
# Welcome

This is your first docs page. Write content in MDX and use components like
<Callout type="info">this</Callout>.
```

## File Structure

Place your docs under `app/docs/`:

```
app/
  docs/
    layout.tsx      # Layout (one line with createDocsLayout)
    page.mdx        # /docs
    introduction/
      page.mdx      # /docs/introduction
    guides/
      advanced/
        page.mdx    # /docs/guides/advanced
```
