# Get Started

Welcome to our documentation framework. This section covers everything you need
to get up and running quickly.

## What You'll Learn

- How to install and configure the framework
- Setting up your first documentation site
- Configuring AI-powered search
- Deploying your docs to production

## Prerequisites

Before getting started, make sure you have:

- **Node.js 18+** installed
- **pnpm** (recommended) or npm/yarn
- A GitHub account for version control
