# Configuration

All configuration lives in a single `docs.config.ts` file at your project root.

## Basic Config

```ts
import { defineDocs } from "@farming-labs/docs";
import { fumadocs } from "@farming-labs/fumadocs";

export default defineDocs({
  entry: "docs",
  theme: fumadocs(),
});
```

## Theme Options

Customize colors, typography, and component defaults:

```ts
theme: fumadocs({
  ui: {
    colors: { primary: "#22c55e" },
    components: { Callout: { variant: "outline" } },
    layout: { toc: { enabled: true, depth: 3 } },
  },
}),
```

## Custom Components

Pass your own React components to use in MDX without imports:

```ts
import { MyNote } from "./app/components/my-note";

export default defineDocs({
  // ...
  components: {
    MyNote,
  },
});
```

Then use `<MyNote>` directly in any `.mdx` file.

## Icon Registry

Map string labels to React elements for sidebar icons:

```tsx
import { BookOpen, Terminal, Rocket } from "lucide-react";

export default defineDocs({
  // ...
  icons: {
    book: <BookOpen size={16} />,
    terminal: <Terminal size={16} />,
    rocket: <Rocket size={16} />,
  },
});
```

Then reference them in page frontmatter:

```yaml
---
title: "CLI Reference"
icon: "terminal"
---
```

## Open Graph

Enable automatic OG image generation:

```ts

  enabled: true,
  type: "dynamic",
  endpoint: "/api/og",
  defaultImage: "/og/default.png",
},
```
