# Get Started

The Next.js example keeps the shared docs UI locale-aware while the top-level
docs routes choose the correct MDX page from `searchParams.lang`.

## What To Check

- Open `/docs/getting-started?lang=en` and `/docs/getting-started?lang=fr`.
- Switch languages from the sidebar select beside the theme toggle.
- Confirm the page content changes with the selected locale.
