# Installation

Install the packages for the Next.js docs example:

```bash
pnpm add @farming-labs/docs @farming-labs/next @farming-labs/theme
```

## Configure Docs

Enable `i18n` in `docs.config.tsx`, then render localized page content from the
route wrapper based on `searchParams.lang`.
