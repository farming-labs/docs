# Themes

Themes control the visual appearance of your documentation site.

## Applying a Theme

```tsx
import { defineDocs } from "@farming-labs/docs";
import { greentree } from "@farming-labs/theme/greentree";

export default defineDocs({
  theme: greentree(),
});
```

## Cross-Framework Theme Parity

All built-in themes, including `monolith`, are available across Next.js, TanStack Start, SvelteKit, Astro, and Nuxt.

```ts
// Next.js / TanStack Start
import { monolith } from "@farming-labs/theme/monolith";

// SvelteKit
import { monolith as svelteMonolith } from "@farming-labs/svelte-theme/monolith";

// Astro
import { monolith as astroMonolith } from "@farming-labs/astro-theme/monolith";

// Nuxt
import { monolith as nuxtMonolith } from "@farming-labs/nuxt-theme/monolith";
```

```css
/* Next.js / TanStack Start */
@import "@farming-labs/theme/monolith/css";

/* SvelteKit */
@import "@farming-labs/svelte-theme/monolith/css";

/* Astro */
@import "@farming-labs/astro-theme/monolith/css";

/* Nuxt */
@import "@farming-labs/nuxt-theme/monolith/css";
```

## Theme Options

Each theme accepts an options object:

```tsx
theme: greentree({
  ui: {
    colors: {
      primary: "#0D9373",
      background: "#FFFFFF",
      foreground: "#1A1A1A",
    },
    radius: "0.5rem",
    sidebar: { style: "default", width: 240 },
    layout: { toc: { style: "default" } },
  },
});
```

## Creating a Custom Theme

```tsx
import { createTheme } from "@farming-labs/theme";

export const myTheme = createTheme({
  name: "my-theme",
  ui: {
    colors: {
      primary: "#6366F1",
      background: "#0F0F10",
    },
  },
});
```

## Color Tokens

| Token             | Description             |
| ----------------- | ----------------------- |
| `primary`         | Primary brand color     |
| `background`      | Page background         |
| `foreground`      | Main text color         |
| `muted`           | Muted background        |
| `mutedForeground` | Muted text              |
| `border`          | Border color            |
| `accent`          | Accent/hover background |
