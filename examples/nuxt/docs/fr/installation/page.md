# Installation

Installez les packages utilises par l'exemple Nuxt:

```bash
pnpm add @farming-labs/docs @farming-labs/nuxt @farming-labs/nuxt-theme
```

## Configuration

Definissez `entry`, `contentDir`, `nav` et `i18n` dans `docs.config.ts`.
