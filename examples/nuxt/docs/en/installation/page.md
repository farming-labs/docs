# Installation

Install the packages for the Nuxt docs example:

```bash
pnpm add @farming-labs/docs @farming-labs/nuxt @farming-labs/nuxt-theme
```

## Configure Docs

Set `entry`, `contentDir`, `nav`, and `i18n` in `docs.config.ts`.
