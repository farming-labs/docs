# Get Started

The Nuxt example keeps content, sidebar links, search, and AI requests in sync
with the active `?lang=` value.

## What To Check

- Switch between `en` and `fr` from the sidebar control.
- Open `/docs/get-started?lang=en` and `/docs/get-started?lang=fr`.
- Confirm the locale stays on navigation and search results.
